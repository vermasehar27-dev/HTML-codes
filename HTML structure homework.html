Python

Python is a high-level, general-purpose programming language known for its simple, English-like syntax and emphasis on code readability. It is a versatile and beginner-friendly language widely used in various fields, including web development, data science, artificial intelligence, and automation.

Scratch

"Scratch" most commonly refers to a free, block-based visual programming language and online community developed by the MIT Media Lab. It is the world’s largest coding community for children and is designed to help young people (primarily ages 8–16)learn to think creatively and reason systematically.

HTML

HTML (HyperText Markup Language) is the standard language for creating and structuring web pages, using tags to define elements like headings, paragraphs, links, and images, telling web browsers how to display content and build the basic framework of any website. It's a foundational technology, working with CSS for styling and JavaScript for interactivity, forming the backbone of the internet.

JavaScript

JavaScript is a versatile, high-level programming language that is one of the core technologies of the World Wide Web, alongside HTML and CSS. It is used to enable dynamic and interactive content on websites, such as animated graphics, interactive maps, and real-time updates.
